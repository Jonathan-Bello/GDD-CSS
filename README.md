# GDD CSS

## Link del proyecto publicado

https://css.jonathanbello.com/

## Descripcion

Sitio web documental para el Game Design Document de CSS. Incluye pagina principal, seccion del GDD, demo web, configuracion para el juego y enlaces a repositorios relacionados.

## Tecnologias usadas

- Astro
- React
- Tailwind CSS
- JavaScript
- Godot
- Backend en Render

## Requisitos

- Node.js 18+
- npm
- Acceso al backend de Hemis para validar la integracion del juego

## Retos tecnicos

- Documentar un GDD navegable y publicarlo como sitio web.
- Conectar la experiencia web con configuracion consumida por el juego.
- Mantener enlaces entre demo, repositorios y backend de forma clara.
- Coordinar contenido tecnico de juego con una presentacion web legible.
