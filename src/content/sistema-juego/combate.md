# Sistema de Combate

## Introducción

Este documento describe el sistema de combate del juego.

## Mecánicas Básicas

### Turnos

- Cada combate se divide en turnos
- Los jugadores y enemigos actúan alternativamente
- El orden se determina por la estadística de velocidad

### Acciones

- **Ataque**: Daño físico al enemigo
- **Defensa**: Reduce el daño recibido
- **Habilidad**: Usa puntos de energía para efectos especiales
- **Objeto**: Usa items del inventario

## Estadísticas

| Stat      | Descripción       |
| --------- | ----------------- |
| Vida      | Puntos de salud   |
| Ataque    | Daño físico       |
| Defensa   | Reducción de daño |
| Velocidad | Orden en combate  |
