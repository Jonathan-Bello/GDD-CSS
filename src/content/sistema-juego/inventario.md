# Sistema de Inventario

## Descripción General

El inventario organiza todos los items que el jugador recolecta durante su aventura en CSS Quest AI.

## Tipos de Items

### Consumibles

Items que se usan y desaparecen del inventario:

- Pociones de salud
- Pociones de energía
- Comida y bebidas
- Catalizadores CSS

### Equipo

Armas, armaduras y accesorios que proporcionan bonificadores:

- Armaduras (cuerpo, cabeza, manos, pies)
- Armas (espadas, arcos, bastones)
- Accesorios (anillos, amuletos, gafas)

### Quest Items

Items especiales necesarios para misiones:

- Llaves temáticas
- Documentos importantes
- Objetos de cutscene

### Recursos

Materiales para crafting y upgrades:

- Mineral de CSS
- Fragmentos de código
- Esencias elementales
- Telas y cuero

## Gestión del Espacio

- Capacidad máxima: 20 espacios
- Cada item ocupa 1 espacio (excepto apilables)
- Los consumibles pueden apilarse hasta 99 unidades
- Sistema de drop para descargar items

## Equipo Activo

El jugador puede equipar hasta 5 items simultáneamente que proporcionan bonificadores de estadísticas.
