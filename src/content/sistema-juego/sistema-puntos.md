# Sistema de Puntos y Experiencia

## Introducción

El sistema de puntos motiva al jugador a completar objetivos y explorar el mundo del juego.

## Ganancia de Experiencia

Los jugadores ganan XP (Experience Points) al:

- Completar puzzles CSS
- Derrotar enemigos
- Completar misiones
- Descubrir secretos
- Interactuar con NPCs

### Tabla de XP Requerida

| Nivel | XP Total | XP Requerida |
| ----- | -------- | ------------ |
| 1     | 0        | 100          |
| 2     | 100      | 150          |
| 3     | 250      | 200          |
| 4     | 450      | 250          |
| 5     | 700      | 300          |

## Puntos de Habilidad

Cada nivel ganado otorga 3 puntos de habilidad que se pueden invertir en:

- Fuerza
- Defensa
- Velocidad
- Inteligencia

## Logros y Recompensas

Los logros especiales dan bonificadores permanentes:

- +10% XP por partida
- Equipamiento especial
- Acceso a áreas restringidas
- Emotes únicos
