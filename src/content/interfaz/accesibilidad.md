## Checklist base (**provisional**)

| Ítem | Estado objetivo | Nota |
|---|---|---|
| Auto-target activable/desactivable | Obligatorio | Opción independiente de dificultad. |
| Ayudas de apuntado | Obligatorio | Incluye priorización de objetivo y retícula clara. |
| Remapeo de controles | Objetivo futuro | Planificar para versión posterior. |
| Tamaño de texto | Obligatorio | Mínimo tres escalas configurables. |
| Velocidad de texto | Obligatorio | Ajuste para diálogo y mensajes de sistema. |
| Contraste | Obligatorio | Presets de alto contraste para HUD/editor. |
| Lectura clara del HUD | Obligatorio | Jerarquía visual estable en combate. |
| Opciones asistidas de dificultad | Evaluar | Reducir castigo sin romper loop base. |

## Criterios de aceptación

- Toda opción de accesibilidad debe ser explicada en lenguaje claro.
- Cambios de accesibilidad no deben requerir reiniciar partida.
- El impacto de cada opción debe ser visible inmediatamente en UI.
