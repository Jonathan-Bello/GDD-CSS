# Menú Principal

## Descripción

El menú principal es la primera interfaz que encuentra el jugador. Presenta las opciones de navegación principales y controles atmosféricos.

## Opciones Disponibles

### Nuevo Juego

Inicia una nueva partida. Si hay una partida guardada, pregunta si desea continuar o sobrescribir.

### Continuar

Carga la última partida guardada. Solo aparece si hay datos de guardado disponibles.

### Cargar Partida

Muestra una lista de guardados disponibles para cargar. Cada guardado muestra:

- Nombre de personaje
- Nivel actual
- Horas jugadas
- Última zona visitada
- Fecha de guardado

### Configuración

Acceso a opciones de:

- Audio (volumen general, música, efectos)
- Video (resolución, fullscreen, gráficos)
- Controles (remapeo de teclado)
- Idioma

### Créditos

Muestra los créditos del desarrollo del juego y agradecimientos especiales.

### Salir

Cierra la aplicación del juego.

## Elementos Visuales

- **Fondo**: Castillo de los Estilos con efecto parallax
- **Música**: Tema épico ambiental
- **Animaciones**: Elementos CSS flotantes sutiles
- **Modo oscuro obligatorio**: Tema consistente con la experiencia
