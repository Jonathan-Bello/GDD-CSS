# Sistema de Misiones

## Tipos de Misiones

### Misiones Principales

Son obligatorias para progresar en la historia. Revelan la narrativa y conducen a nuevas áreas.

- Número: ~15-20 misiones principales
- Duración: 2-3 horas de juego total
- Recompensas: XP abundante, items únicos, acceso a nuevas áreas

### Misiones Secundarias

Opcionales pero recompensadoras. Proporcionan contexto adicional y profundidad al mundo.

- Número: ~30+ misiones secundarias
- Duración: 15-30 minutos cada una
- Recompensas: XP moderado, items especiales, historias laterales

### Misiones Diarias

Desafíos menores que se resetean cada día en el juego.

- Nuevo desafío de CSS cada día
- Recompensas: XP, moneda especial
- Acumulable para bonificadores

## Estructura de Quest

Cada misión contiene:

1. **Activador**: Hablando con un NPC o encontrando un objeto
2. **Objetivo Principal**: Lo que debe hacer el jugador
3. **Objetivos Secundarios**: Tareas adicionales para recompensas extras
4. **Recompensas**: XP, items, moneda
5. **Conclusión**: Dialogar con el NPC para completar

## Seguimiento de Progreso

El diario de misiones muestra:

- Misiones activas
- Misiones completadas
- Misiones disponibles
- Historial de logros

## Ramificaciones de Quest

Algunas misiones tienen múltiples soluciones que afectan el final del juego:

- Solución violenta vs pacífica
- Ayudar a diferentes facciones
- Diferentes resultados narrativos
