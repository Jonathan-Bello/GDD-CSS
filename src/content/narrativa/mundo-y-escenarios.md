# El Mundo de CSS Quest AI

## Estructura General

El mundo de CSS Quest AI está dividido en varias regiones temáticas, cada una representando diferentes aspectos del CSS y el diseño web.

## Castillo de los Estilos

La región inicial donde despierta el jugador. Un majestuoso castillo flotante donde las paredes responden a los principios de CSS.

- Ambiente: Mágicomedieval
- Enemigos: Sprites de overflow y elementos rotos
- Jefe de Región: Kron el Corrupto

## Bosque de Selectores

Un bosque misterioso donde los árboles son selectores CSS vivos. Cada selector tiene su propia lógica y comportamiento.

- Ambiente: Naturaleza fusionada con código
- Enemigos: Especificidad malvada, selectores recursivos
- Jefe de Región: El Selector Primario

## Océano de Layouts

Aguas ondulantes que representan los diferentes sistemas de layout: float, flexbox, y grid.

- Ambiente: Marino y fluido
- Enemigos: Clearing floats, contenedores colapsados
- Jefe de Región: Maestra de las Mareas

## Cordillera de Variables

Montañas donde se almacenan todas las variables CSS y temas del juego.

- Ambiente: Picos nevados y cuevas de datos
- Enemigos: Variables modificadas, conflictos de scope
- Jefe de Región: El Guardián de Valores

## Metrópolis de Componentes

Una ciudad moderna llena de componentes reutilizables y patrones de diseño.

- Ambiente: Futurista y ordenado
- Enemigos: Componentes rotos, dependencias circulares
- Jefe de Región: La Reina de la Modularidad
