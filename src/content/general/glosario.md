## Objetivo

Este glosario define términos oficiales de _Citadel of Solar Souls (IA)_ para evitar ambigüedad en producción.

| Término | Definición canónica |
|---|---|
| **Pilar solar** | Espacio seguro donde el jugador recupera **Luz Solar**, guarda progreso y puede editar munición CSS. Término oficial (equivalente funcional a una “hoguera”). |
| **Cápsula de desafío** | Microespacio de prueba donde se enseña y valida una propiedad CSS concreta. Otorga desbloqueos de progresión. |
| **Slot de munición** | Espacio equipable que guarda una configuración de bala CSS. Se cambia en tiempo real, pero solo se edita en un Pilar solar. |
| **Sinergia CSS** | Relación entre propiedades/valores de la bala y metadata CSS del enemigo que modifica el daño final. |
| **Luz Solar** | Recurso limitado de curación del jugador. Se consume por carga y se recarga en un Pilar solar. |
| **Hemis** | IA diegética acompañante. Brinda apoyo contextual y aprendizaje de CSS durante la partida. |
| **Xanat** | Protagonista y avatar silencioso del jugador. Núcleo de exploración, combate y progresión narrativa. |
| **Propiedad desbloqueada** | Propiedad CSS habilitada oficialmente por progresión. Puede usarse en cálculo de munición y sinergia. |
| **Auto-target** | Sistema opcional de accesibilidad para fijado/priorización de objetivo válido en combate. |
| **Compendio CSS** | Registro consultable de propiedades desbloqueadas, usos prácticos y notas de aplicación. |

## Nota de uso editorial

Cuando exista terminología histórica en documentos antiguos (por ejemplo, “zona segura” o “zona de control”), debe traducirse al término oficial: **Pilar solar**.
